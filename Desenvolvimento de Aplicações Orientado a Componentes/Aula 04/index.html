Sobre

Este site foi criado por mim, usando o conceito de SPA (single page application).

Um SPA é uma aplicação web que roda em uma única página, sem recarregar a página inteira a cada interação do usuário.

Isso torna a navegação mais rápida e fluida, além de economizar recursos do servidor.
